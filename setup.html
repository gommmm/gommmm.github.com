설치하기

설치 요구조건
  1. PHP 5.3.7 이상 필요
  2. mod_rewrite 필요
  3. GD Library 2.0 이상 필요
  4. Node.js 필요
설치 방법
  1. 보드 설치 소스를 github 페이지에 들어가서 다운로드 받습니다.
    2. github 페이지
    github 페이지에서 를 클릭하고 클릭
  2. 다운받은 파일을 원하는 웹서버 폴더에 압축풀기
  3. node 폴더에서 server.js 가동
  4. 리눅스나 맥에 서버를 두고 있으면 최상위 폴더의 파일권한을 707이나 777로 변경
  5. 압축을 푼 폴더가 예를 들어 board이면

    1. application/config/constants.php 90번째 줄 SUB_DIRECTORY 상수에 경로 추가

      define('SUB_DIRECTORY', '') => define('SUB_DIRECTORY', '/board')

    2. resource/plugin/seditor/sample/photo_uploader/file_uploader_html5.php 31번째 줄 변수에 경로 추가

      $sub = '/board';

    3. resource/plugin/seditor/sample/photo_uploader/attach_photo.js 336번째 줄 경로 추가

      var sub = "" => var sub = "/board"

  6. 저장하고 설치하고자 하시는 웹사이트 주소를 입력하면 설치하라는 페이지가 나오는데 설치를 눌러줍니다.
  7. 그러면 데이터베이스 정보 및 관리자 정보를 입력하는 폼이 나오는데 입력해줍니다.
  8. 데이터베이스 정보를 올바르게 입력하면 설치가 완료됐다는 화면이 나오고 설치가 완료됩니다.
  9. 이제 모두 설치를 완료했습니다. 메인페이지로 링크를 눌러줍니다.
